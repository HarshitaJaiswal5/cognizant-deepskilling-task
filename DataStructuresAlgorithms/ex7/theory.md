Recursion is a programming technique in which a method calls itself to solve a problem.

It simplifies problems by breaking them into smaller subproblems until a base condition is reached.

Analysis
Time Complexity
Time Complexity: O(n), where n is the number of years.
The recursive method makes one recursive call for each year.
Optimization

The recursive solution can be optimized by:

Using Memoization to store previously computed values and avoid repeated calculations.
Using an Iterative (loop-based) approach, which avoids recursion overhead and is more memory-efficient.
